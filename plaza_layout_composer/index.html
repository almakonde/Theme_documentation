Installation

Copy or upload all folders, files in "Source Code" folder to your root website folder

Now, you can use this module immediately. It's very simple to install.

Module Initialization

  • Login your opencart admin

  • Go to PlazaThemes Dashboard -> Modules -> Plaza Layout Composer Click Install button
  • When the page load successfully with Inform text: "Success: You have modified modules!", click Edit button

Setting Layout Builder module

Let me explain all parts of this module

  1. Layout Name: The name of content which you will add to your Layouts in Design / Layouts

  2. Status: This content is shown when the status is enabled.

  3. Plus / Minus Button: These buttons can divide multiple columns in a row.

    All columns have a same width. The maximum of columns is 12 based on the bootstrap columns format.

    2 columns

    3 columns

    4 columns

    6 columns

    12 columns

  4. Specified Class: Add a class to a specific row

  5. Columns Customization / Delete Row Buttons

    • Columns Custom Button

      This button allows you custom columns whatever you want (based on bootstrap formart)

      When you click this button, a popup will be shown

      The popup above has number 12. It means there are one column in a row.

      You must fill this text follow the plus format: "num1 + num2 + num3 + ..."

      And also, the total always must be 12

      Example 1: We set "4 + 2 + 4 + 2" and click OK (Total is 12)

      The row will be divided by 4 columns with ratio of columns width: 4/12 - 2/12 - 4/12 - 2/12

      Example 2: We set "2 + 4 + 6" and click OK (Total is 12)

      The row will be divided by 3 columns with ratio of columns width: 2/12 - 4/12 - 6/12

    • Delete Row Button

      You can delete row by clicking this button.

  6. Edit / Delete Module Button

    • Edit Module

      When you click this button, the module settings will be shown and you can configure this module.

      Example: We click this button of Plaza Slider > Home slideshow 1 module at the previous step

    • Delete Module

      When you click this button, the module will be deleted.

  7. Add Modules Button: Each columns have this button. It allows you add modules for each columns.

    When you click this button, all extensions/modules will be shown in the popup.

    You can choose extensions/modules you want to add into column.

    Example:

    • Click Add Modules button

    • Choose module Carousel > BrandSlider.

    • Now, the Carousel > BrandSlider is in the first column.

  8. Add Row Button: When you click this button, the new row will be added and you can modify this row by yourself.

* Note: With Plaza Layout Composer module, you can drag/drop rows and modules, of course.

Setting Design > Layout

Now, we have "Store 1 - Home Page" at the above steps.

Go to Design -> Layouts, click Store 1 - Home Page Layout layout, then click Edit button . (you can choose other layouts to set other pages)

You just add "Store 1 - Home Page" module and Save

Now, all settings have shown in the homepage correctly.
